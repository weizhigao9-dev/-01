## 2. 视频选题简介
**视频标题**: 当 AI 拥有了身体：带你秒懂具身智能
**选题方向**: 什么是具身智能？与传统 AI 有何不同？
**视频时长**: 约 3 分钟
**内容摘要**: 
本短视频以通俗易懂的方式向大众科普了“具身智能（Embodied AI）”的概念。视频首先通过与 ChatGPT 等传统“缸中之脑”AI 的对比，点明了具身智能的核心区别在于拥有与物理世界交互的“身体”；随后详细解释了具身智能在感知、规划与控制方面的三大核心技术；最后展望了智能机器人未来的应用场景，探讨了具身智能将如何改变人类的生活方式。视频采用了“出镜讲解 + 画面混剪展示”的形式，结构遵循“引入悬念—原理解释—总结展望”的三段式叙事。
